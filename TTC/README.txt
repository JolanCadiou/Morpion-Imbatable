TTC1.0 est un Morpion qui possède un mode 1 joueur et un mode 2 joueur.
Le mode 1 joueur possède une IA imbattable.
Celle-ci est entièrement de ma conception et ne correspond donc sûrement à aucun algorithme existant.
Le fichier Test.txt sert de base de données pour enregistrer les scores des parties.
